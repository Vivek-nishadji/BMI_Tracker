**BMI Tracker Application**

**Overview**

The BMI Tracker Application is a web-based tool designed to help users calculate and monitor their Body Mass Index (BMI). Built using Django, this application provides a simple and user-friendly interface for entering weight and height to compute BMI, categorize health status, and save records for future reference.

Features
	•	BMI Calculation: Input weight and height to calculate BMI.
	•	Health Status Categorization: Categorizes BMI results into underweight, normal, overweight, or obese.
	•	Record Management:
	•	View BMI history.
	•	Delete records if needed.
	•	Interactive User Interface: A responsive and visually appealing interface for easy navigation.

 Technology Stack
	•	Backend Framework: Django
	•	Frontend: HTML5, CSS3, Bootstrap (optional for styling)
	•	Database: SQLite (default in Django, can be replaced with MySQL or PostgreSQL)
	•	Language: Python
